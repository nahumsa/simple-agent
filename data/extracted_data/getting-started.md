# Getting Started

## Steps
1. Visit https://codingchallenges.fyi/challenges/intro/ to browse all available challenges.
2. Pick a challenge that interests you - beginners should start with the wc Tool or JSON Parser challenges.
3. Choose your preferred programming language - all challenges are language agnostic.
4. Follow the step-by-step instructions in the challenge. Each step builds on the previous one.
5. Share your solution on GitHub and tag John Crickett on Bluesky or LinkedIn.

## Recommended First Challenges
- Build Your Own wc Tool - a gentle introduction to CLI tools and file processing
- Build Your Own JSON Parser - learn parsing fundamentals applicable to many domains
- Build Your Own URL Shortener - a classic web application project
- Build Your Own Calculator - straightforward parsing and evaluation

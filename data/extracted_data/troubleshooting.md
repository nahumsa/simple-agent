# Troubleshooting

If you get stuck on a challenge, try these steps in order before giving up:

## Steps
1. Re-read the challenge description and the current step carefully - the requirement is often more specific than it first looks.
2. Search Google or your preferred search engine for the specific error message, protocol detail, or algorithm you're stuck on.
3. Ask a friend or colleague - explaining the problem out loud often surfaces the answer.
4. Visit the Coding Challenges Discord server to ask the community.
5. Look at example shared solutions at https://github.com/CodingChallengesFYI/sharedsolutions to see how other developers approached the same challenge.

**Difficulty:** intermediate

**URL:** https://codingchallenges.fyi/challenges/challenge-docker

**Skills:** Linux namespaces, cgroups, system calls, container fundamentals, Docker Registry API

## Description

Build a lightweight Docker clone that uses Linux namespaces, cgroups, and chroot to create isolated containers. Pull and run images from Docker Hub.

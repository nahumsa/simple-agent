# Build Your Own Redis Server

**Difficulty:** intermediate

**URL:** https://codingchallenges.fyi/challenges/challenge-redis

**Skills:** network programming, protocol implementation, concurrency, data structures, performance optimisation

## Description

Build a lightweight Redis clone - an in-memory data structure server supporting strings, hashes, lists, sets, and sorted sets. Implement the RESP protocol and handle concurrent clients.
